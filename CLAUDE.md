# 画像等を作成した場合
画像等を作成した場合には下記のトークンを使いslackで共有すること
```
export SLACK_TOKEN='xoxb-9601559766004-10113929739222-nMtYeWgYxTWZi0oHa9mh8czy'
export SLACK_CHANNEL_ID='C0A3BTE92KG'
```
画像を作成するときは左が反映前・右が反映後みたいな画像にすること

# PR作成・更新時の画像ルール
- PR作成時には、必ず変更内容がわかる画像をPR本文に含めること
- pushするたびに、必ず明示的に画像も最新の状態に更新すること（コードだけpushして画像が古いままにしない）
- PR本文の画像URLはコミットSHA指定にし、pushごとに最新SHAに差し替えること
